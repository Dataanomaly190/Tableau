# Tableau Data Visualization Portfolio

This repository contains a collection of interactive dashboards developed using **Tableau Desktop**. These projects focus on data storytelling, exploratory data analysis (EDA), and visual aesthetics to derive meaningful insights from complex datasets.

## Projects Overview

### 1. World Happiness Index Analysis (2015 - 2019)

A comprehensive multi-year study of global happiness trends.

- **Files:** `Happiness Index (2015-2019) - DashBoard.twbx`
- **Key Insights:** Tracking shifts in economic production, social support, life expectancy, freedom, absence of corruption, and generosity across 150+ countries.
- **Visuals:** Map distributions, trend lines, and factor correlation matrices.

### 2. World Cup Analysis

Deep dive into FIFA World Cup history and match statistics.

- **Files:** `World Cup - Dashboard.twbx`, `World Cup Matches - DashBoard.twbx`
- **Key Insights:** Historical performance of nations, goal scoring patterns, and match-level analytics.
- **Visuals:** Tournament history dashboards, player/team performance metrics, and stadium attendance data.

## How to View

To interact with these dashboards, you will need **Tableau Desktop** or **Tableau Public** (free version).

1. Clone this repository:

   ```bash
   git clone https://github.com/Dataanomaly190/FullStackProjects.git
   ```

2. Navigate to the `TableauProjects` folder.
3. Open any `.twbx` file to launch the dashboard.

## Tools Used

- **Tableau Desktop** for visualization.
- **Excel/CSV** for data sourcing.
- **Power Query / Python (Pandas)** for any necessary pre-processing.
